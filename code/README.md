# Cloned Repositories

## Repo 1: modelling-consciousness
- **URL**: https://github.com/Xaxis/modelling-consciousness
- **Purpose**: Formal system for quantifying and modelling consciousness; aims to be rigorous, testable, and universal
- **Location**: code/modelling-consciousness/
- **Key files**: README.md (contains the theoretical framework)
- **Relevance**: Attempts to create a measurable framework for consciousness - relevant to whether distinctions between theories can be operationalized into testable metrics

## Repo 2: iit-consciousness
- **URL**: https://github.com/virgil/consciousness
- **Purpose**: Computing effective information (ei), minimum information partition (MIP), integrated information (phi), and other IIT measures
- **Location**: code/iit-consciousness/
- **Key files**: consc/ (main library), tests/ (test cases), balduzzi_python/ (Python implementation)
- **Relevance**: Implementation of IIT's core computational measures. Relevant for examining whether IIT's mathematical formalism produces behaviorally testable predictions (the "unfolding argument" challenges this).

## Repo 3: consciousness-framework
- **URL**: https://github.com/KarolFilipKowalczyk/Consciousness
- **Purpose**: Comprehensive theoretical framework integrating IIT, GWT, AST, and quantum consciousness approaches
- **Location**: code/consciousness-framework/
- **Key files**: main.tex and part*.tex files (LaTeX manuscript), appendix_comparisons.tex (theory comparison), bibliography.bib (references), gwt/ and selectors/ (implementations)
- **Relevance**: Includes systematic comparison of major theories and testable experimental protocols. The appendix_comparisons.tex file may contain structured comparisons relevant to identifying "costume" vs substantive theoretical differences.

## Notes
- These repositories provide computational tools and theoretical frameworks that could support analysis of whether consciousness theories make meaningfully different predictions
- The IIT implementation (Repo 2) is particularly relevant for testing the Doerig et al. "unfolding argument" - showing that systems with identical behavior can have different phi values
